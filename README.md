<h3>Projeto Pessoal: Desenvolvimento de Pipeline de Análise de Dados</h3>

<div align = "justify">
Projeto pessoal desenvolvido durante o curso de Programação em Python para Data Science do SENAI. Consistiu na criação de um pipeline completo de dados, abrangendo etapas de coleta, tratamento e análise utilizando Pandas. O projeto envolveu a limpeza e transformação de dados brutos em conjuntos estruturados, permitindo a construção de dashboards interativos para visualização de informações. Como resultado, foi possível identificar padrões, tendências e gerar insights relevantes para tomada de decisão orientada por dados.
</div>

<h3>Ferramentas Utilizadas:</h3>
<b>Python</b>


<b>Pandas</b>  
<b>Google Colab</b>
